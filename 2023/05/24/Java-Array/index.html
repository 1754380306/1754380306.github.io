Java数组学习

数组，是一种用于存储多个相同类型数据的存储模型

定义格式：

  • 数据类型[] 数组名；
  • 举例：int[] arr;

数组初始化：

  1. 所谓初始化，就是维数组中的数组元素分配内存空间，并为每个数组元素赋值
  2. 可分为动态初始化和静态初始化

动态初始化：

  1. 初始化时只指定数组长度，由系统为数组分配初始值
  2. 格式：数据类型[] 变量名 = new 数据类型[数组长度]；
  3. 范例:int[] arr = new int[3];

数组元素访问

数组变量用数组名访问

数组内储存的元素用数组名[索引]访问

索引:

  • 从0开始
  • 连续的
  • 每次加一
